# AndroidUtils
安卓工具集合


### about KeyboardUtils
```
registerKeyboardChangeListener  ：软键盘高度变化监听
```

### about ConvertUtils
```
dp2px  ：dp转px
px2dp  ：px转dp
sp2px  ：sp转px
px2sp  ：px转sp
```


### about ScreenUtils
```
getScreenHeight         : 获取屏幕高度
getScreenWidth          ：获取屏幕宽度
getNavigationBarHeight  ：获取导航栏的高度
getStatusBarHeight      ：获取状态栏的高度
```

### about ZipUtils
```
unZip  ：解压
```
